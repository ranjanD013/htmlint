Research :
We have collected all the information, like what kind of campaign Customers want to set. Also searched about other donation sites, the design trends. Then draw all the ideas & try to make the main focus of this site, like how a user can complete their journey in a seamless way possible.

Scenario 1:
*We need to set up the campaigns with all its information, so the user can read about it & easily donate on that one. The main points should be there, so customers can see the progress of it & get interested to join other roles.
*From our childhood, we have known about row-column theory. Anyone can understand much better when it appears. So on that note I can use this one for setting up the campaigns.

Solution

We tried to create the campaign boxes with a digital draw-in column, so it is easily understandable (every main info will be there) & the contribution process will be easier.
*From our childhood, we have known about row-column theory. Anyone can understand much better when it appears. So on that note I can use this one for setting up the campaigns.

Scenario 2:
*We need to showcase the donor name so customers can motivate others to contribute.
*We need to showcase the donor name so customers can motivate others to contribute.

Solution:

We have created a chart with top & recent donors, from where people can see how many customers are active on donations. Time and money amount also will be there for more info.

Solution:
Added location filters in the campaign page, so anyone can find it easily, from where the campaign has been raised.

Wireframes:

Low fidelity wireframes are shared with customers, and reviewed and reworked before these are given to Addy Digital Salesforce Implementation Services.
*In this wireframe, we have shown the entire campaign page. Where users can see all the campaigns easily.
*We have added some important filters, so the user can shortlist easily, which he/she needs.

Wireframes:

*After clicking a campaign, this page will pop up. Where you can read all the information about that particular campaign.
*The Contribute now button is highlighted, which will catch people’s eye fast.
*Below the contribution this donation chart is added to show the donor's history.

Donation:

*Below the contribution this donation chart is added to show the donor's history.
*Via this donation form the user can contribute easily & an email will be sent to his/her ID as notification.
*After donation this Thank You page will pop up. If you want to know more about other roles, that link with the chevron icon will redirect there.
